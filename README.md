Repositorio del desarrollo de un miliohmetro para la medición de resistencias de bajo valor, basado en el módulo ESP32-WROOM-32 y una aplicación móvil desarrollada en MIT App Inventor.

Incluye:

* Implementación del sistema de medición utilizando el integrado ESP32-WROOM-32
* Código para adquisición de señales y cálculo de resistencia
* Aplicación móvil para visualización en tiempo real
* Simulaciones en Proteus (medición Kelvin, amplificador de instrumentación y módulo de carga)
* Diseño de PCB en EasyEDA
* Pruebas y validación del sistema

El proyecto integra simulación, diseño electrónico y desarrollo de software como etapa previa a la implementación de un instrumento de medición de precisión.
